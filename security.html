Web security

geolocation

Geolocation API allows a user to provide their location if they desire with permission to report locations information. Longitude and latitude can be acquired and is acurrate with in a few feet.

platform

This property returns which platform the browser is compiled for, such as Windows or mac OS in the form of Win32 or MacIntel.

online

The online property returns a value that can tell wheather the browser is online or offline

appName

The appName property returns the name of the browser that is being used.

userAgent

The userAgent property returns the value of the user-agent header sent by the browser to the server. The value returned, contains information about the name, version and platform of the browser.

appVersion

The appVersion property returns the version information of the browser.

Get Safe Online

Going online has various risks associated with visiting malicious websites or disclosure of personal information. Risks of this nature can inclued malware viruses to fraud from fake websites phising for personal information. Being safe online is up to the end user in being weiry of uncommon sites using common sense and making sure that the sites visited are secure with a lock and secure with "htpps://" before giving over personal information.

How to stay safe on the web

Keeping your software/OS up to date can protect you from vulnerabilitys by providing patches that protect your computer and personal information. Best practices to use to protect your information range from creating secure passwords, deciding on what information to share, and being conscious about your online security.

10 Tips To Stay Safe Online

Create complex strong unique passwords and use multi-factor authentication. Make sure the conection that you use to get online is secure, if you use public wifi use a VPN to secure your connection. Use a firewall to limit what comes in and out of your connection. Be selective in what you share online and make sure you your software is up todate to prevent vulnerability hacks.
